Planilha .PSC

A planilha .PSC é um banco de dados referente aos processos de prestação de serviços comunitários, armazena os dados do prestador, pena, entidade e horas prestadas, os dados são primordialmente inseridos via API pelo site da 4º vara.

A planilha é implementada através da lógica de sub planilhas, cada entidade registrada possui uma planilha própria referenciada na aba 'Quadro de Vagas', a planilha de cada entidade é organizada através de abas, cada prestador possui uma aba com seu nome, a qual é alterada pela entidade. Os dados exibidos no site, PDFs gerados e na aba 'Lotação' da referente planilha nada mais são do que um reflexo da aba de um prestador preenchida por uma entidade.

Portanto, é de responsabilidade das funções presentes nesta planilha:

  • Envio de e-mails automáticos para informar entidades sobre a situação dos prestadores
  • Troca de entidades de prestador (o site apenas sinaliza que a troca deve ser feita, através da coluna 'Temporário'), a planilha deve realizar a troca e normalizar os dados.
  • Geração de PDF das abas de prestadores através da biblioteca 'PDF' importada.
  • Exclusão do prestador (também requerida pelo site através da aba 'Temporário'), a planilha deve normalizar os dados na planilha de entidade referente e mover o prestador para o registro de excluídos.
  • Criar planilhas de entidade através da aba 'Cadastro Entidade'.
  • Criar abas de prestadores na respectiva planilha de entidades quando um prestador é adicionado via site.
